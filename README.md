# Comment Wall - Frontend

A responsive web interface for posting and managing comments. This is the frontend portion of the Comment Wall project.

## Features

- **View Comments:** Automatically fetches and displays recent comments with dynamic timestamps (e.g., "Hace 5 minutos").
- **Post Comments:** Quick and easy comment creation with built-in character limit validation.
- **Delete Comments:** Remove unwanted comments with a single click and a safety confirmation prompt.

## Tech Stack

- HTML
- CSS (Custom Dark Theme)
- Vanilla JavaScript (Fetch API)
- Deployment: **Vercel**

## Live Demo

- **Frontend:** https://commentwall.vercel.app/
- **Backend API:** Hosted on Render
